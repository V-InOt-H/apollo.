Project Overview

Redacted is an enterprise-grade calibration tracking system built for Tier-1 automotive manufacturers. The platform automates the entire calibration lifecycle — from scheduling and tracking to automated reminders and compliance reporting.

The system integrates with existing industrial IoT devices to pull real-time calibration data, automatically flagging instruments that are due for recalibration. Custom notification workflows ensure that technicians receive timely alerts via email, SMS, and in-app notifications.

Built with Rust for performance-critical components and FastAPI for the web interface, the system handles 10,000+ calibration records with sub-100ms query times. The system reduced missed calibrations by 94% and saved an estimated 2,400 man-hours annually.
